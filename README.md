# Fiado (v2)

> Controle de vendas a prazo ("fiado") para pequenos comerciantes — reescrita
> do [FiadoApp](https://github.com/drizaogythub97/FiadoApp) com Next.js,
> TypeScript e Supabase, no padrão do [Gaveta](https://github.com/drizaogythub97/gaveta).

**Status:** em desenvolvimento (F1 — fundação). Roadmap em
[`docs/01-ROADMAP-FASES.md`](docs/01-ROADMAP-FASES.md).

## Stack

Next.js (App Router) · TypeScript · Tailwind CSS + shadcn/ui ·
Supabase (PostgreSQL + Auth + RLS) · Vercel · Vitest + Playwright · Zod

## Ecossistema

O Fiado faz parte do mesmo ecossistema do **Gaveta** (ERP de frente de caixa):
apps independentes, mesma conta (SSO via Supabase Auth) e navegação cruzada.
